# Snake
This is a simple Snake game. It was created with Python and the "pygame" module

# How it works
1. Start the "game.py" file
2. Play

# Controls
## Movement
Left Arrow - Move left<br>
Right Arrow - Move right<br>
Up Arrow - Move up<br>
Down Arrow - Move Down<br>

## Game functionality
p - Pause/Unpause
